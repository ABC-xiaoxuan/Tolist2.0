# ToList Desktop 1.1.0

本次更新重点补齐任务提醒能力，并继续优化待办浮窗体验。

- 新增任务到期时间，可在任务高级选项中设置。
- 新增到期提醒方式：准时提醒、提前 5 分钟、提前 30 分钟、自定义时间提醒。
- 新增系统通知插件接入，到达提醒时间后通过桌面通知提醒。
- 新增本地提醒调度逻辑，主窗口统一调度，避免浮窗重复触发提醒。
- 任务列表与待办浮窗新增到期时间和提醒方式标签。
- SQLite 新增 `reminder_at`、`due_at`、`reminder_mode` 字段迁移，兼容旧任务数据。
- 优化待办浮窗展开动画，减少跳变感。
- 修复待办浮窗透明外框残留问题。
